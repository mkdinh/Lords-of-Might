# Lords of Mights

An browser MMO RPG blending classic RPG elements with a modern UI

# Technologies
- MaterializeCSS
- ExpressJS
- PhaserJS
- MySQL
- Socket.IO

# Features
- Create your own character
- Level progression
- Multiplayer turn-based battle
- In-game world and private chat
